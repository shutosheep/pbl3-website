Taxi Go

by Manami Yano

  • Features:
    • This app shows real-time location information of taxis registered with Taxi Go. This app is geolocation By using the taxi go app, it is possible to call a taxi to a specified location, set the destination and complete the payment with the app, and reduce the number of interactions in the car. Calling a taxi has become easier because we don't have to look up the phone number to call a taxi or pay for the phone.
  • Pros:
    • It is easy to set pick-up and drop-off locations.
    • It is possible to reduce interactions in the car.
    • It can be known a taxi price in advance.
    • It is possible to choose a taxi company and vehicle.
    • It is possible to see the pick-up fee for each taxi company.
    • It is easy to find when users forget something in taxi.
  • Cons:
    • Pick-up fee/arrangement fee will be charged.
    • Coupons can only be used with GO Pay.
    • Some areas are not available.
